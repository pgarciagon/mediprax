Suizidalitaetseinschaetzung¶

Die strukturierte Suizidalitaetseinschaetzung ist ein zentrales Instrument der psychiatrischen Dokumentation. MediPrax bietet eine systematische Erfassung mit Risikofaktoren, Schutzfaktoren und Sicherheitsplanung.

Risikostufen¶

Die Einschaetzung erfolgt in 5 Stufen:

Stufe Bezeichnung Systemreaktion
0 Kein Risiko Keine besondere Markierung
1 Geringes Risiko Dokumentation im Kontakt
2 Mittleres Risiko Warnbanner in der Patientenakte
3 Hohes Risiko Prominentes Warnbanner, systemweit sichtbar
4 Akutes Risiko Prominentes Warnbanner, systemweit sichtbar

Warnbanner

Ab Risikostufe Mittel (2) wird in der gesamten Patientenakte ein auffaelliges Warnbanner angezeigt. Dieses ist fuer alle Nutzer sichtbar und stellt sicher, dass das erhoehte Risiko bei jedem Patientenkontakt wahrgenommen wird.

Einschaetzung dokumentieren¶

  1. Oeffnen Sie einen Kontakt oder die Patientenakte.
  2. Waehlen Sie Suizidalitaetseinschaetzung.
  3. Dokumentieren Sie folgende Bereiche:

Strukturierte Erfassung¶

Bereich Beschreibung
Suizidgedanken Art, Haeufigkeit und Intensitaet suizidaler Gedanken
Suizidplaene Vorhandensein konkreter Plaene, Methode
Suizidabsicht Absicht zur Umsetzung
Fruehere Suizidversuche Anzahl, Zeitpunkt, Methode, Schweregrad
Aktuelle Handlungen Vorbereitende Handlungen

Risikofaktoren (20 Items)¶

MediPrax stellt einen Katalog von 20 Risikofaktoren bereit, z. B.:

  • Fruehere Suizidversuche
  • Psychische Erkrankung (Depression, Schizophrenie, Suchterkrankung)
  • Chronische Schmerzen / schwere koerperliche Erkrankung
  • Aktuelle Krisensituation (Trennung, Verlust, Arbeitslosigkeit)
  • Soziale Isolation
  • Zugang zu Mitteln
  • Hoffnungslosigkeit
  • Impulsivitaet
  • Substanzmissbrauch
  • Maennliches Geschlecht, hoeheres Alter

Schutzfaktoren (14 Items)¶

Ergaenzend werden 14 Schutzfaktoren dokumentiert, z. B.:

  • Tragende soziale Beziehungen
  • Kinder / Verantwortung fuer Angehoerige
  • Religioese / spirituelle Ueberzeugungen
  • Therapeutische Beziehung
  • Absprachefaehigkeit
  • Zukunftsplaene
  • Behandlungsmotivation

Sicherheitsplan¶

Bei erhoehtem Risiko kann ein Sicherheitsplan dokumentiert werden:

  • Warnzeichen erkennen
  • Individuelle Bewaeltigungsstrategien
  • Soziale Kontakte zur Ablenkung
  • Professionelle Hilfe (Telefonnummern)
  • Umgebung sichern (Zugang zu Mitteln einschraenken)

Massnahmen¶

Dokumentieren Sie die ergriffenen Massnahmen, z. B.:

  • Engmaschige Terminvergabe
  • Medikationsanpassung
  • Einweisung (freiwillig oder nach PsychKG)
  • Krisenintervention
  • Angehoerigenkontakt

Tipp

Dokumentieren Sie die Suizidalitaetseinschaetzung bei jedem Kontakt mit suizidgefaehrdeten Patienten. Die lueckenlose Dokumentation schuetzt auch forensisch.

Hinweis

Fuer eine Einweisung nach PsychKG nutzen Sie das PsychKG-Formular zur Erstellung des aerztlichen Zeugnisses.
