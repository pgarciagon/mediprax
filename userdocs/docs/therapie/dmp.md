# DMP Depression

Das Disease Management Programm (DMP) Depression ermoeglicht die strukturierte Dokumentation und Verlaufsbeobachtung depressiver Patienten gemaess den Anforderungen der Kassenaerztlichen Vereinigung.

## Patienten einschreiben

1. Oeffnen Sie die [Patientenakte](../patienten/akte.md).
2. Navigieren Sie zum Bereich **DMP Depression**.
3. Klicken Sie auf **Patient einschreiben**.
4. Bestaetigen Sie die Einschreibung und das Startdatum.

!!! info "Voraussetzung"
    Die Einschreibung setzt eine dokumentierte Diagnose aus dem Bereich F32/F33 (Depressive Episode / Rezidivierende depressive Stoerung) voraus.

## Quartalsdokumentation

Die DMP-Dokumentation erfolgt **quartalsweise**. Fuer jedes Quartal erfassen Sie:

| Feld | Beschreibung |
|---|---|
| **PHQ-9-Score** | Aktueller Depressions-Schweregrad (siehe [Psychometrie](../dokumentation/psychometrie.md)) |
| **Medikationsstatus** | Aktuelle antidepressive Medikation, Aenderungen |
| **Psychotherapiestatus** | Laufende Psychotherapie (Ja/Nein), Art und Haeufigkeit |
| **Komorbiditaeten** | Relevante Begleiterkrankungen |
| **Hospitalisierung** | Stationaere Behandlung im Quartal (Ja/Nein) |
| **Behandlungsziele** | Aktuelle therapeutische Ziele |
| **Zielerreichung** | Bewertung der Zielerreichung |

## Schritt-fuer-Schritt Quartalsdokumentation

1. Oeffnen Sie die DMP-Dokumentation des Patienten.
2. Klicken Sie auf **Neue Quartalsdokumentation**.
3. Waehlen Sie das **Quartal** (z. B. Q1/2026).
4. Fuehren Sie den **PHQ-9** durch oder uebernehmen Sie einen aktuellen Wert.
5. Dokumentieren Sie Medikation, Therapie und Verlauf.
6. Klicken Sie auf **Speichern**.

## Uebermittlung an die KV

Nach Abschluss der Quartalsdokumentation:

1. Pruefen Sie die Vollstaendigkeit aller Pflichtfelder.
2. Markieren Sie die Dokumentation als **Fertig**.
3. Die Uebermittlung an die KV erfolgt ueber den [KVDT-Export](../abrechnung/kvdt.md).

## Uebermittlungsstatus

| Status | Beschreibung |
|---|---|
| **Offen** | Dokumentation noch nicht vollstaendig |
| **Fertig** | Alle Pflichtfelder ausgefuellt |
| **Uebermittelt** | An die KV gesendet |
| **Bestaetigt** | Von der KV akzeptiert |

!!! tip "Tipp"
    Fuehren Sie den PHQ-9 standardmaessig bei jedem Kontakt mit DMP-Patienten durch. So haben Sie am Quartalsende immer einen aktuellen Wert verfuegbar.

!!! warning "Wichtig"
    Die Quartalsdokumentation muss fristgerecht uebermittelt werden. Achten Sie auf die Einreichfristen der Kassenaerztlichen Vereinigung.
