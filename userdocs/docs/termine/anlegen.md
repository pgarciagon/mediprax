# Termin anlegen

Neue Termine erstellen Sie ueber die Terminplanung. Navigieren Sie zu **Termine** > **Neuer Termin** oder oeffnen Sie direkt `/termine/neu`.

## Schritt-fuer-Schritt

1. **Patient auswaehlen**: Beginnen Sie mit der Eingabe des Patientennamens im Suchfeld. Die **Autovervollstaendigung** schlaegt passende Patienten vor. Waehlen Sie den gewuenschten Patienten aus.
2. **Arzt/Aerztin waehlen**: Waehlen Sie den behandelnden Arzt aus der Dropdown-Liste.
3. **Datum festlegen**: Waehlen Sie das Termindatum ueber den Kalender.
4. **Uhrzeit eingeben**: Geben Sie die Startuhrzeit ein.
5. **Dauer festlegen**: Waehlen Sie die Dauer in Minuten (wird automatisch vorausgefuellt, wenn eine Terminart gewaehlt wurde).
6. **Terminart waehlen** (optional): Waehlen Sie eine [Terminvorlage](vorlagen.md), um die Dauer automatisch zu setzen.
7. **Notizen** (optional): Fuegen Sie interne Notizen hinzu.
8. Klicken Sie auf **Speichern**.

## Terminart-Vorlagen

Waehlen Sie eine Terminart, um die Dauer automatisch vorzubelegen:

| Terminart | Standarddauer |
|---|---|
| Erstgespraech | 50 Minuten |
| Folgetermin | 25 Minuten |
| Medikamentenkontrolle | 10 Minuten |
| Psychotherapie-Sitzung | 50 Minuten |
| EEG | 30 Minuten |
| EMG/NLG | 45 Minuten |
| Akuttermin | 25 Minuten |
| Notfall | 15 Minuten |

!!! info "Hinweis"
    Die Dauer kann nach Auswahl der Terminart jederzeit manuell angepasst werden. Die Vorlage dient als Vorschlag.

## Termin bearbeiten

1. Klicken Sie auf den Termin in der [Wochenansicht](wochenansicht.md).
2. Aendern Sie die gewuenschten Felder.
3. Klicken Sie auf **Speichern**.

## Termin loeschen

1. Oeffnen Sie den Termin.
2. Klicken Sie auf **Loeschen**.
3. Bestaetigen Sie die Loeschung im Dialog.

!!! warning "Wichtig"
    Geloeschte Termine koennen nicht wiederhergestellt werden. Wenn ein Patient nicht erschienen ist, markieren Sie den Termin stattdessen als **Nicht erschienen** im [Wartezimmer](wartezimmer.md).

!!! tip "Tipp"
    Fuer regelmaessig wiederkehrende Termine nutzen Sie die [Terminserien](serien.md) -- damit entfaellt das wiederholte manuelle Anlegen.
