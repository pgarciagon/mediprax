# Willkommen bei MediPrax

MediPrax ist ein modernes **Praxisverwaltungssystem (PVS)**, das speziell fuer **psychiatrische und neurologische Praxen** entwickelt wurde. Es unterstuetzt Sie im gesamten Praxisalltag -- von der Patientenaufnahme ueber die klinische Dokumentation bis zur Quartalsabrechnung.

## Funktionsuebersicht

| Bereich | Beschreibung |
|---|---|
| **Patientenverwaltung** | Patientenstammdaten, Versicherungsinformationen, Betreuer |
| **Terminplanung** | Wochenansicht, Terminserien, Wartezimmer, Warteliste |
| **Klinische Dokumentation** | Psychopathologischer Befund (AMDP), neurologischer Befund, Psychometrie, Suizidalitaetseinschaetzung, ABDTP-Sektionen, Aktionsketten, Textbausteine |
| **Krankheitsspezifische Module** | Anfallskalender, Kopfschmerz-Tagebuch, MS-Dokumentation, Parkinson-Dokumentation |
| **Therapieverwaltung** | Therapiefaelle, PTV-Formulare, DMP Depression |
| **Medikation** | Medikamentenverwaltung, Labormonitoring, BtM-Verwaltung, Rezeptdruck |
| **Abrechnung** | EBM/GOAe, Plausibilitaetspruefung, GOP-Vorschlaege, Quartalsvalidierung, KVDT-Export, Privatrechnungen |
| **Formulare** | Arztbriefe, KBV-Formulare, PsychKG-Dokumentation, eAU |
| **Geraeteanbindung** | GDT-Schnittstelle (EEG, EMG, NLG), Videosprechstunde |
| **Telematikinfrastruktur** | eGK-Lesung, E-Rezept, KIM-Kommunikation |
| **Verwaltung** | Benutzerverwaltung, Textbausteine, Aktionsketten, Datenimport, Audit-Protokoll |

## Schnellstart

Steigen Sie schnell in die wichtigsten Funktionen ein:

1. **[Anmeldung](erste-schritte/anmeldung.md)** -- Einloggen und erste Orientierung
2. **[Dashboard](erste-schritte/dashboard.md)** -- Tagesuebersicht und anstehende Aufgaben
3. **[Patient anlegen](patienten/anlegen.md)** -- Neuen Patienten erfassen
4. **[Termin erstellen](termine/anlegen.md)** -- Ersten Termin vergeben
5. **[Kontakt dokumentieren](dokumentation/kontakte.md)** -- Behandlung dokumentieren und abrechnen
6. **[Quartalsvalidierung](abrechnung/quartalsvalidierung.md)** -- Abrechnung pruefen und einreichen

!!! tip "Tipp fuer den Einstieg"
    Beginnen Sie mit der [Anmeldung](erste-schritte/anmeldung.md) und machen Sie sich mit dem [Dashboard](erste-schritte/dashboard.md) vertraut. Von dort aus navigieren Sie ueber die Seitenleiste zu allen Modulen.

## Zielgruppe

Dieses Handbuch richtet sich an alle Nutzer von MediPrax:

- **Aerzte/Aerztinnen** -- Vollzugriff auf alle klinischen und administrativen Funktionen
- **MFA (Medizinische Fachangestellte)** -- Terminverwaltung, Patientenaufnahme, Formulare
- **Empfang** -- Terminvergabe, Wartezimmerverwaltung, eGK-Lesung

!!! info "Hinweis"
    Die verfuegbaren Funktionen richten sich nach Ihrer Benutzerrolle. Nicht alle in diesem Handbuch beschriebenen Funktionen stehen jeder Rolle zur Verfuegung.

## Systemvoraussetzungen

MediPrax ist eine webbasierte Anwendung und laeuft in jedem modernen Browser (Chrome, Firefox, Edge, Safari). Eine separate Installation ist nicht erforderlich. Fuer die Telematikinfrastruktur-Anbindung werden ein Konnektor, ein eGK-Kartenlesegeraet und eine SMC-B-Karte benoetigt.
