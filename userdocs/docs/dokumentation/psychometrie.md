# Psychometrische Tests

MediPrax unterstuetzt die Durchfuehrung und Auswertung standardisierter psychometrischer Testverfahren. Die Ergebnisse werden automatisch berechnet, klassifiziert und im Verlauf dargestellt.

## Verfuegbare Tests

| Test | Bereich | Items | Scorebereich |
|---|---|---|---|
| **PHQ-9** | Depression | 9 Items | 0-27 |
| **GAD-7** | Angststoerungen | 7 Items | 0-21 |
| **BDI-II** | Depression | 21 Items | 0-63 |
| **MMST** | Kognition (Demenz-Screening) | 30 Items | 0-30 |
| **MoCA** | Kognition (MCI-Screening) | 30 Items | 0-30 |

## Test durchfuehren

1. Oeffnen Sie die [Patientenakte](../patienten/akte.md).
2. Navigieren Sie zum Bereich **Psychometrie**.
3. Klicken Sie auf **Neuer Test**.
4. Waehlen Sie das gewuenschte **Testverfahren**.
5. Fuellen Sie die einzelnen **Items** aus.
6. Der **Gesamtscore** wird automatisch berechnet.
7. Klicken Sie auf **Speichern**.

## Schweregradklassifikation

### PHQ-9 (Depression)

| Score | Schweregrad |
|---|---|
| 0-4 | Keine Depression |
| 5-9 | Leichte Depression |
| 10-14 | Mittelgradige Depression |
| 15-19 | Mittelschwere Depression |
| 20-27 | Schwere Depression |

### GAD-7 (Angst)

| Score | Schweregrad |
|---|---|
| 0-4 | Keine Angststoerung |
| 5-9 | Leichte Angst |
| 10-14 | Mittelgradige Angst |
| 15-21 | Schwere Angst |

### MMST (Kognition)

| Score | Einstufung |
|---|---|
| 27-30 | Keine kognitive Beeintraechtigung |
| 20-26 | Leichte kognitive Beeintraechtigung |
| 10-19 | Maessige kognitive Beeintraechtigung |
| 0-9 | Schwere kognitive Beeintraechtigung |

## Verlaufsbeobachtung

MediPrax speichert alle Testergebnisse chronologisch und ermoeglicht eine **laengsschnittliche Verlaufsbeobachtung**:

- Grafische Darstellung des Scoreverlaufs ueber die Zeit
- Erkennung von Trends (Verbesserung/Verschlechterung)
- Vergleich mit Vortestergebnissen

!!! tip "Tipp"
    Fuehren Sie den PHQ-9 bei Depressionspatienten regelmaessig (z. B. alle 4 Wochen) durch. Der Verlauf dokumentiert den Therapieerfolg und ist wertvolle Grundlage fuer Berichte und DMP-Dokumentation.

!!! info "Hinweis"
    Die Testergebnisse koennen fuer die [DMP-Depression-Dokumentation](../therapie/dmp.md) herangezogen und in [Arztbriefe](../formulare/arztbriefe.md) uebernommen werden.
