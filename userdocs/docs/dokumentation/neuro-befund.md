# Neurologischer Befund

Der strukturierte neurologische Befund dokumentiert die koerperlich-neurologische Untersuchung. Er wird bei neurologischen Patienten als Teil der Kontaktdokumentation erhoben.

## Befund erstellen

1. Oeffnen Sie einen [Kontakt](kontakte.md) oder erstellen Sie einen neuen.
2. Waehlen Sie den Bereich **Neurologischer Befund**.
3. Dokumentieren Sie die Untersuchungsbefunde in den einzelnen Kategorien.
4. Der Befundtext wird **automatisch generiert**.
5. Speichern Sie den Befund.

## Untersuchungskategorien

| Kategorie | Untersuchte Bereiche |
|---|---|
| **Hirnnerven** | N. I-XII: Visus, Gesichtsfeld, Pupillomotorik, Augenmotilitaet, Sensibilitaet Gesicht, Fazialismotorik, Gehoer, Schlucken, Zunge |
| **Motorik** | Muskeltonus, Muskelkraft (KG 0-5), Trophik, Muskeleigenreflexe, pathologische Reflexe |
| **Sensibilitaet** | Oberflaechensensibilitaet, Tiefensensibilitaet, Temperaturempfinden, Schmerzempfinden |
| **Reflexe** | BSR, TSR, RPR, PSR, ASR; Pyramidenbahnzeichen (Babinski, Gordon, Oppenheim) |
| **Koordination** | Finger-Nase-Versuch, Knie-Hacke-Versuch, Diadochokinese, Romberg-Stehversuch |
| **Gang** | Gangbild, Seiltaenzergang, Einbeinstand, Unterberger-Tretversuch |

## Bewertung

Die Befunde werden strukturiert erfasst mit:

- **Seitenangabe**: Rechts, Links, Beidseits
- **Auspraegung**: Normal, Leicht auffaellig, Deutlich pathologisch
- **Numerische Scores**: Z. B. Kraftgrade 0-5 nach MRC-Skala

| Kraftgrad | Beschreibung |
|---|---|
| 0 | Keine Muskelkontraktion |
| 1 | Sichtbare Kontraktion ohne Bewegungseffekt |
| 2 | Bewegung unter Ausschaltung der Schwerkraft |
| 3 | Bewegung gegen die Schwerkraft |
| 4 | Bewegung gegen Widerstand |
| 5 | Normale Kraft |

## Automatische Textgenerierung

Aus den strukturierten Eingaben wird ein **klinischer Befundtext** generiert, z. B.:

> *Hirnnerven regelrecht. Keine latenten oder manifesten Paresen. Muskeltonus und Trophik unauffaellig. MER seitengleich mittellebhaft auslösbar. Keine Pyramidenbahnzeichen. Sensibilitaet fuer alle Qualitaeten intakt. Koordination und Gang unauffaellig.*

Dieser Text kann direkt in Arztbriefe uebernommen werden.

!!! tip "Tipp"
    Fuer einen Normalbefund koennen Sie alle Kategorien auf "Normal" belassen. Der generierte Text beschreibt dann automatisch einen unauffaelligen neurologischen Status.

!!! info "Hinweis"
    Der neurologische Befund ergaenzt den [psychopathologischen Befund](psychopath-befund.md). Bei psychiatrisch-neurologischen Kombinationspraxen koennen beide Befunde in einem Kontakt erhoben werden.
