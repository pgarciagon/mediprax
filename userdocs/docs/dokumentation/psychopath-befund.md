# Psychopathologischer Befund

Der psychopathologische Befund nach dem **AMDP-System** ermoeglicht eine strukturierte Erfassung psychischer Symptome. Er wird als Teil der klinischen Dokumentation bei psychiatrischen Patienten erhoben.

## Befund erstellen

1. Oeffnen Sie einen [Kontakt](kontakte.md) oder erstellen Sie einen neuen.
2. Waehlen Sie den Bereich **Psychopathologischer Befund**.
3. Bewerten Sie die einzelnen Symptomkategorien.
4. Der Befundtext wird **automatisch generiert**.
5. Speichern Sie den Befund.

## Symptomkategorien

Die Erhebung umfasst folgende AMDP-Kategorien:

| Kategorie | Beispiel-Items |
|---|---|
| **Bewusstsein** | Bewusstseinslage, Bewusstseinsklarheit |
| **Orientierung** | Zeitlich, oertlich, situativ, zur Person |
| **Aufmerksamkeit** | Konzentration, Auffassungsgabe |
| **Gedaechtnis** | Kurzzeitgedaechtnis, Langzeitgedaechtnis |
| **Formales Denken** | Denkhemmung, Gedankenabreissen, Ideenflucht, Perseveration |
| **Inhaltliches Denken** | Wahn, Zwangsgedanken, ueberwertige Ideen |
| **Wahrnehmung** | Halluzinationen (akustisch, optisch, taktil), Illusionen |
| **Ich-Stoerungen** | Depersonalisation, Derealisation, Gedankeneingebung |
| **Affektivitaet** | Stimmung, Schwingungsfaehigkeit, Angst, Insuffizienzgefuehle |
| **Antrieb** | Antriebslage, Interesse, Energie |
| **Psychomotorik** | Mimik, Gestik, Motorische Unruhe, Stupor |
| **Zirkadiane Rhythmik** | Schlaf, Appetit, Tagesrhythmik |
| **Suizidalitaet** | Siehe [Suizidalitaetseinschaetzung](suizidalitaet.md) |

## Schweregradskala

Jedes Symptom wird auf einer Skala von **0 bis 3** bewertet:

| Wert | Bedeutung |
|---|---|
| **0** | Nicht vorhanden |
| **1** | Leicht ausgepraegt |
| **2** | Maessig ausgepraegt |
| **3** | Schwer ausgepraegt |

## Automatische Textgenerierung

Basierend auf Ihren Eingaben generiert MediPrax automatisch einen **Befundtext in Prosaform**, der die dokumentierten Auffaelligkeiten zusammenfasst. Dieser Text kann:

- In Arztbriefe uebernommen werden
- Als Verlaufsdokumentation dienen
- Bei Bedarf manuell ergaenzt werden

!!! tip "Tipp"
    Nutzen Sie den automatisch generierten Text als Grundlage fuer Ihre Arztbriefe. Er spart erheblich Zeit bei der Befunddokumentation und stellt Vollstaendigkeit sicher.

!!! info "AMDP-System"
    Das AMDP-System (Arbeitsgemeinschaft fuer Methodik und Dokumentation in der Psychiatrie) ist der Standard fuer die psychopathologische Befunderhebung im deutschsprachigen Raum.

!!! note "Hinweis"
    Der psychopathologische Befund kann bei jedem Kontakt neu erhoben werden. Der Verlauf ueber mehrere Befunde hinweg zeigt die Symptomveraenderung im Therapieverlauf.
